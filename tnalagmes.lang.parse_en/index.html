extract_datetime_en

def extract_datetime_en(string, currentDate)

Convert a human date reference into an exact datetime

Convert things like "today" "tomorrow afternoon" "next Tuesday at 4pm" "August 3rd" into a datetime. If a reference date is not provided, the current local time is used. Also consumes the words used to define the date returning the remaining string. For example, the string "what is Tuesday's weather forecast" returns the date for the forthcoming Tuesday relative to the reference date and the remainder string "what is weather forecast".

Args: string (str): string containing date words currentDate (datetime): A reference date/time for "tommorrow", etc

Returns: [datetime, str]: An array containing the datetime and the remaining text not consumed in the parsing.
